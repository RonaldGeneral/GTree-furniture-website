Only four lines of code to copy in your program


Put these two in the first two lines of head tag
Put your webpage name for the tab name inside semicolon of function head("")
<script src="/fixed-layout/fixed-layout.js" type="text/javascript"></script>
<script>head("");</script>


Put this in the first line of body tag
<script>body();</script>


Create an article tag yourself and put your content code in it
If you have article tag already then no need to do it again


Put this in the last line of body tag
<script>footer();</script>



Example:
<!DOCTYPE html>
<html lang="en">
    <head>
        <script src="/fixed-layout/fixed-layout.js" type="text/javascript"></script>
        <script>head("GTree Company");</script>
    </head>
    <body>
        <script>body();</script>

        <article>    
            <!--Your code starts here-->
        </article>

        <script>footer();</script>
    </body>
</html>
